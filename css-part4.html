CSS Cascading

Specificity, importance, source order. !important > inline > id > class > element.

Universal selector (*)

* { margin:0; } /* applies to all */

ID selector

#unique { color: #1e6f9f; }

Group selector

h1, .special, #title { font-weight: 600; }

Class selector

.box { border-radius: 20px; }

Child selector (>)

div > p { color: navy; } /* direct child */

Element selector

p { line-height: 1.5; }

Descendant selector (space)

article p { } /* any p inside article */

General sibling (~)

h2 ~ p { } /* all p after h2 */

Adjacent sibling (+)

h2 + p { } /* p immediately after h2 */
