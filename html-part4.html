📌 HTML head elements

The <head> section contains metadata and links to external resources – not displayed on the page.

<title> – page title

<title>My awesome page</title>

<meta> – metadata

<meta charset="UTF-8">
<meta name="description" content="Free tutorials">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="John Doe">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="refresh" content="30">
<meta name="robots" content="noindex, nofollow">

<base> – base URL for relative links

<base href="https://www.tutorialspoint.com/" target="_blank">

<link> – external resources (CSS, favicon)

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico" type="image/x-icon">

<style> – internal CSS

<style>
  body { background: lightblue; }
</style>

<script> – JavaScript (internal/external)

<script src="script.js"></script>
<script>
  function hello() { alert('hi'); }
</script>

⭐ Favicon (favorite icon)

A small icon displayed in the browser tab. Add using <link rel="icon"> inside <head>.

<link rel="icon" type="image/png" href="favicon.png">
<link rel="icon" type="image/x-icon" href="favicon.ico">

Formats: ICO, PNG, GIF, SVG. Common sizes: 16x16, 32x32, 48x48.

⚡ JavaScript in HTML

Three ways to include JavaScript:

  • Inline: <button onclick="alert('hi')">
  • Internal: inside <script> ... </script>
  • External: <script src="file.js"></script>

Example: internal script

<script>
  function showMessage() {
    alert('Hello from internal script');
  }
</script>
<button onclick="showMessage()">Click</button>

<noscript> – fallback for disabled JS

<noscript>Your browser does not support JavaScript.</noscript>

📐 HTML layouts – semantic elements

Use semantic tags to structure your page: <header>, <nav>, <section>, <article>, <aside>, <footer>. 

<body>
  <header>Header</header>
  <nav>Navigation</nav>
  <section>
    <article>Main content</article>
    <aside>Sidebar</aside>
  </section>
  <footer>Footer</footer>
</body>

Layout using CSS float

nav { float: left; width: 20%; }
article { float: left; width: 80%; }

Layout using CSS flexbox

.container { display: flex; }
nav { flex: 1; }
article { flex: 4; }

Layout using CSS grid

.grid { display: grid; grid-template-columns: 1fr 4fr; }

📱 Responsive web design

Make pages adapt to different screen sizes.

Viewport meta tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Responsive images

<img src="image.jpg" style="max-width:100%; height:auto;">

Media queries

@media screen and (max-width: 600px) {
  body { background: lightblue; }
  .sidebar { display: none; }
}

Responsive text (vw unit)

<h1 style="font-size:6vw;">Responsive heading</h1>

📝 HTML style guide (best practices)

  • Always declare <!DOCTYPE html>
  • Specify document language: <html lang="en">
  • Set character encoding: <meta charset="UTF-8">
  • Use lowercase for elements and attributes
  • Quote attribute values (double quotes)
  • Close all tags (including optional ones like </p>)
  • Use proper indentation (2 or 4 spaces)
  • Add comments: <!-- section explanation -->
  • Set viewport for mobile: <meta name="viewport" ...>

🎨 SVG – scalable vector graphics

Define vector graphics using XML. Embed with <svg> or <img>.

Basic shapes

<svg width="200" height="200">
  <circle cx="50" cy="50" r="40" fill="red" />
  <rect x="10" y="10" width="80" height="60" fill="blue" />
  <line x1="0" y1="0" x2="100" y2="100" stroke="green" stroke-width="2" />
  <ellipse cx="100" cy="100" rx="50" ry="30" fill="purple" />
  <polygon points="20,10 300,20 170,50" fill="orange" />
  <polyline points="0,0 0,30 30,30 30,60" fill="none" stroke="black" />
</svg>

Gradient example

<svg>
  <defs>
    <linearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" style="stop-color:red;" />
      <stop offset="100%" style="stop-color:blue;" />
    </linearGradient>
  </defs>
  <rect x="10" y="10" width="150" height="80" fill="url(#grad)" />
</svg>

🖼️ HTML canvas – drawing with JavaScript

<canvas> creates a drawing surface. Use JavaScript to draw.

<canvas id="myCanvas" width="300" height="200" style="border:1px solid #ccc;"></canvas>
<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');

  // rectangle
  ctx.fillStyle = 'red';
  ctx.fillRect(20, 20, 100, 80);

  // circle
  ctx.beginPath();
  ctx.arc(150, 100, 40, 0, 2*Math.PI);
  ctx.fillStyle = 'blue';
  ctx.fill();

  // line
  ctx.beginPath();
  ctx.moveTo(10, 10);
  ctx.lineTo(200, 150);
  ctx.strokeStyle = 'green';
  ctx.stroke();

  // text
  ctx.font = '20px Arial';
  ctx.fillStyle = 'black';
  ctx.fillText('Hello canvas', 30, 180);
</script>

Canvas methods overview

  • fillRect(x, y, w, h) – filled rectangle
  • strokeRect(x, y, w, h) – outlined rectangle
  • clearRect(x, y, w, h) – clear area
  • beginPath(), moveTo(), lineTo(), stroke() – draw lines
  • arc(x, y, r, startAngle, endAngle) – circle/arc
  • quadraticCurveTo(), bezierCurveTo() – curves
  • fillText(), strokeText() – text
  • drawImage() – place images
  • createLinearGradient(), createRadialGradient() – gradients
  • translate(), rotate(), scale() – transformations
  • save(), restore() – save/restore state
