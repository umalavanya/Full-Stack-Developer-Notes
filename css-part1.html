CSS Tutorial

Job Search · PDF Version · Quick Guide · Resources · Discussion

This CSS tutorial covers everything from basic styling concepts and selectors to advanced techniques, such as flexbox, grid, animations, and CSS variables. This CSS tutorial is designed to help both beginners and experienced designers to make them masters in creating visually appealing, responsive, and modern web designs.

What is CSS

CSS is the acronym for "Cascading Style Sheet". It's a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS helps the web developers to control the layout and other visual aspects of the web pages. CSS plays a crucial role in modern web development by providing the tools necessary to create visually appealing, accessible, and responsive websites.

CSS Versions

Since the inception of CSS, several versions have came into existence. Some of the notable versions include:

  • CSS1 – The initial version, released in December 1996. Basic styling.
  • CSS2 – Released in May 1998, introduced positioning, z-index, media types.
  • CSS2.1 – W3C Recommendation June 2011, refined CSS2.
  • CSS3 – Collection of modules: selectors, columns, animations, gradients, etc.
  • CSS4 (in progress) – Ongoing effort to extend CSS3.

Each version of CSS builds upon the previous ones. CSS is referred as just CSS now, without a version number.

Types of CSS

1. Inline CSS: Applied directly via style attribute. Highest priority.

<p style="color: blue; font-size: 16px;">This line is an inline-styled paragraph.</p>

2. Internal CSS: Defined within <style> in <head>.

<head> <style> p { color: green; font-size: 18px; } </style> </head>

3. External CSS: Separate .css file linked via <link>.

<head> <link rel="stylesheet" href="styles.css"> </head>

Advantages of Using CSS

  • Responsive design – media queries, adapt to different screens.
  • Flexibility and Control – precise layout and typography.
  • Consistency and Reusability – central file, easy updates.
  • Search Engine Optimization (SEO) – better structure.
  • Ease of Maintenance – global changes.
  • Faster Page Loading – caching external CSS.

Components of CSS

A CSS rule contains a selector and a declaration block.

CSS Example

<style>
  h1 { color: #36CFFF; }
  p { font-size: 1.5em; color: white; }
  div { border: 5px inset gold; background-color: black; width: 300px; text-align: center; }
</style>

In the above: h1, p, div are selectors; color, font-size etc. are properties.
