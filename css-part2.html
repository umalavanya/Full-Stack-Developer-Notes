CSS visibility Property

Previous ◀ Quiz ▶ Next

The visibility property allows you to show or hide an element without changing the layout. Hidden elements still take up space.

Value Description
visible default, element visible
hidden hides element but retains space
collapse for table rows/cols, removes like hidden
initial, inherit standard
🔷 visible
🔶 hidden (invisible but space)
collapse (non-table = hidden)
<style> h2 { visibility: hidden; } </style>

CSS overflow Property

Controls how content overflowing its box is handled.

overflow: visible; long unbroken content spills out

overflow: hidden; long unbroken content disappears

overflow: scroll; scroll to see hidden part

CSS Clearfix

Without clearfix container collapses. With .clearfix::after { clear:both; } parent contains floats.

CSS float Property

float: left; text wraps around. Clear both to stop wrapping.
