💻 HTML computer code elements

Special tags for displaying code, keyboard input, variables, and preformatted text.

<kbd> – keyboard input

<p>Press <kbd>Ctrl</kbd>+<kbd>C</kbd> to copy.</p>

Result: Press Ctrl+C to copy.

<pre> – preformatted text

<pre>
  This text
    preserves spaces
      and line breaks.
</pre>

Result:

  This text
    preserves spaces
      and line breaks.

<code> – code snippet

<p>The <code>function()</code> keyword defines a function.</p>

Result: The function() keyword defines a function.

<var> – variable

<p>The area is <var>πr²</var></p>

Result: The area is πr²

<samp> – sample output

<p><samp>Error: file not found</samp></p>

Result: Error: file not found

🔍 HTML meta tags

<meta> tags provide metadata about the document – they don't display but are used by browsers, search engines, and other services.

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Free tutorials">
  <meta name="keywords" content="HTML, CSS, JavaScript">
  <meta name="author" content="John Doe">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="refresh" content="30">
</head>
Attribute Example Purpose
charset charset="UTF-8" character encoding
name="description" content="..." page description (search results)
name="keywords" content="html, css" SEO keywords (mostly ignored now)
name="author" content="..." document author
name="viewport" content="width=device-width, initial-scale=1" responsive design
http-equiv="refresh" content="5; url=https://..." auto‑refresh / redirect
http-equiv="cookie" content="..." set cookies (old, rarely used)

Meta for social media (Open Graph)

<meta property="og:title" content="Title">
<meta property="og:description" content="Desc">
<meta property="og:image" content="image.jpg">

🏷️ HTML class & id

class attribute

Used to classify elements (non‑unique, can be shared).

<style>.highlight { background: yellow; }</style>
<p class="highlight">This is highlighted.</p>
<p class="highlight">Also highlighted.</p>

id attribute

Unique identifier for a single element.

<div id="header">Main header</div>
<script> document.getElementById('header') </script>

Difference

  • id must be unique per page; used for JavaScript and anchor links.
  • class can be reused; mainly for CSS styling.

🖼️ HTML backgrounds

Old way: background attribute (deprecated). Modern: CSS background.

Background color

<body style="background-color: #f0f0f0;"> ... </body>

Background image

<div style="background-image: url('pattern.jpg'); background-repeat: repeat; background-position: center;">
  Content
</div>

Background repeat & position

background-repeat: repeat-x;   /* horizontal repeat */
background-repeat: repeat-y;   /* vertical repeat */
background-repeat: no-repeat;  /* no repeat */
background-position: top left; /* default top left */

📊 HTML tables – advanced

Tables are built with <table>, <tr> (rows), <td> (cells), <th> (headers).

Basic table

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>25</td></tr>
</table>

Table with caption, thead, tbody, tfoot

<table border="1">
  <caption>Employee data</caption>
  <thead>
    <tr><th>ID</th><th>Name</th></tr>
  </thead>
  <tbody>
    <tr><td>1</td><td>John</td></tr>
    <tr><td>2</td><td>Jane</td></tr>
  </tbody>
  <tfoot>
    <tr><td colspan="2">Two employees</td></tr>
  </tfoot>
</table>

Colgroup & col

<colgroup>
  <col span="2" style="background: lightgrey;">
  <col style="background: lightblue;">
</colgroup>

Nested tables

<table border="1">
  <tr>
    <td>Main cell
      <table border="1">
        <tr><td>inner 1</td><td>inner 2</td></tr>
      </table>
    </td>
  </tr>
</table>

Table styling (CSS zebra stripes, hover)

tr:nth-child(even) { background: #f2f2f2; }
tr:hover { background: #ddd; }

📝 HTML lists

Unordered list

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Marker types: disc, circle, square (via CSS list-style-type).

Ordered list

<ol type="1" start="3">
  <li>Third</li>
  <li>Fourth</li>
</ol>

type values: 1 (numbers), A (uppercase letters), a (lowercase), I (Roman), i (small Roman).

Definition list

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>

Nested lists

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
</ul>

🔗 HTML links (hyperlinks)

Links are created with <a> (anchor) tag.

Basic text link

<a href="https://example.com" target="_blank">Visit example</a>

target attribute

  • _self – same tab (default)
  • _blank – new tab/window
  • _parent – parent frame
  • _top – full body of window

Link to a page section (id)

<a href="#section2">Jump to section 2</a>
...
<h2 id="section2">Section 2</h2>

Image as link

<a href="page.html">
  <img src="button.jpg" alt="click me">
</a>

Email link (mailto)

<a href="mailto:someone@example.com?subject=Hello&body=Hi">Send email</a>

Download link

<a href="file.zip" download="custom-name.zip">Download zip</a>

Base path

Use <base> in <head> to set a default URL for all relative links. 

<base href="https://www.tutorialspoint.com/">
<a href="/html/index.htm">HTML Tutorial</a>

Image maps (client‑side)

<img src="map.jpg" usemap="#workmap">
<map name="workmap">
  <area shape="rect" coords="34,44,270,350" href="page.html">
  <area shape="circle" coords="200,100,40" href="other.html">
  <area shape="poly" coords="140,20,190,80,100,80" href="poly.html">
</map>

Coordinate shapes: rect (x1,y1,x2,y2), circle (centerX,centerY,radius), poly (x1,y1,x2,y2,...).

Server‑side image map (deprecated)

<a href="process.map"><img src="image.jpg" ismap></a>

Coordinates are sent to server – rarely used today.
