1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a page using elements enclosed in tags. HTML is not a programming language – it's a markup language that tells the browser how to display content.

  • HyperText – links that connect pages
  • Markup Language – uses tags to define elements (headings, paragraphs, links...)
Why learn HTML? It's the foundation of every website. Without HTML you cannot build web pages. It's essential for front‑end, back‑end, and full‑stack developers.

Your first HTML page

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Save as my.html and open in any browser.

HTML page structure explained

  • <!DOCTYPE html> – tells browser it's HTML5
  • <html> – root element
  • <head> – meta info, title, links to CSS/JS
  • <title> – page title (browser tab)
  • <body> – visible content

Basic tags (headings, paragraphs, breaks)

<h1>Main heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph with a<br>line break.</p>
<hr> <!-- horizontal rule -->

Comments

<!-- this is a comment ––>

2. HTML elements & tags

An element consists of an opening tag, content, and a closing tag. Some elements are void (no content, e.g. <img>, <br>).

<p>This is a paragraph element.</p>
<img src="photo.jpg" alt="description"> <!-- void element -->

Nested elements

<div>
  <p>A paragraph inside a <strong>div</strong>.</p>
</div>

Block vs inline

  • Block: start on new line, take full width (<div>, <p>, <h1>)
  • Inline: flow within text (<span>, <a>, <img>)

3. HTML attributes

Attributes provide additional information about an element. They go inside the opening tag.

<tag attribute="value">content</tag>
Attribute Example Purpose
id <div id="header"> unique identifier
class <p class="highlight"> CSS class
style style="color:red;" inline CSS
src <img src="image.jpg"> source for media
href <a href="link.html"> hyperlink reference
alt alt="description" alternate text (image)
title title="tooltip" extra info (tooltip)
dir dir="rtl" text direction

Boolean attributes (no value needed): disabled, required, readonly, checked.

4. Headings & paragraphs

Headings <h1>–<h6>

<h1>Most important</h1>
<h2>Section title</h2>
<h6>Lowest level</h6>

Use headings to create a document outline – important for SEO and accessibility.

Paragraphs <p>

<p>This is a paragraph. Browsers add space before/after.</p>

You can combine with <br> (line break) and <hr> (thematic break).

6. Quotations & citations

  • <q> – short inline quote (adds quotes)
  • <blockquote> – long block‑level quote (indented)
  • <cite> – title of a work
  • <abbr> – abbreviation (with title attribute)
  • <address> – contact info
  • <bdo> – override text direction
<q>Short quote</q>
<blockquote cite="source-url">Longer quote...</blockquote>
<cite>Book title</cite>
<abbr title="HyperText Markup Language">HTML</abbr>
<address>123 Main St, City</address>
<bdo dir="rtl">This text is right-to-left.</bdo>

7. Lists

Unordered list <ul>

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Ordered list <ol>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

You can nest lists, change marker type with type="1/A/a/I/i", start with start="5".

8. Images & image maps

<img src="path.jpg" alt="description" width="300" height="200">

Supported formats: JPEG, PNG, GIF, SVG, WebP. Use alt for accessibility.

Image maps (<map> + <area>)

<img src="map.jpg" usemap="#workmap">
<map name="workmap">
  <area shape="rect" coords="34,44,270,350" href="page.html" alt="area">
  <area shape="circle" coords="200,100,40" href="other.html">
  <area shape="poly" coords="140,20,190,80,100,80" href="poly.html">
</map>

Shapes: rect (x1,y1,x2,y2), circle (centerX,centerY,radius), poly (x1,y1,x2,y2,...).

9. Frames (legacy) & iframes

Note: <frameset> and <frame> are deprecated in HTML5. Use <iframe> for embedding external content.

Iframe – inline frame

<iframe src="page.html" width="500" height="300" title="description">
  Fallback text (if not supported)
</iframe>

You can target an iframe with a link using target="frameName" and name="frameName".

Common attributes: allow, loading="lazy", sandbox, referrerpolicy.

10. Using CSS with HTML

Three ways:

  • Inline – style="color:blue;"
  • Internal – <style> ... </style> inside <head>
  • External – <link rel="stylesheet" href="style.css">
<head>
  <style>
    p { color: red; }
  </style>
  <link rel="stylesheet" href="global.css">
</head>

11. Colors & fonts

Colors can be specified by name (red), hex (#ff0000), rgb (rgb(255,0,0)), hsl.

Fonts: use CSS properties font-family, font-size, font-weight. The <font> tag is obsolete.

12. Semantic HTML (HTML5)

Use elements that describe meaning, not just layout:

<header> <nav> <main> <article> <section> <aside> <footer>

Improves accessibility, SEO, and code readability.

13. HTML editors & resources

Popular editors: VS Code, Sublime, Notepad++, Brackets, Atom. Online playgrounds: CodePen, JSFiddle, Tutorialspoint online HTML editor.

2025 learning roadmap: start with basics → lists/links/images → forms → semantic HTML → accessibility → integration with CSS/JS. Practice by building small pages.

14. Quick FAQ

  • Is HTML a programming language? No, it's a markup language.
  • Latest version? HTML5 (living standard).
  • Difference <head> and <header>? <head> contains meta‑data; <header> is a semantic section for introductory content.
  • How to save an HTML file? Use .html extension.
