🎨 RGB & RGBA colors

rgb(red, green, blue) – each value 0..255. rgba(red, green, blue, alpha) adds opacity (0 = transparent, 1 = opaque).

<p style="color: rgb(255,0,0);">Red text</p>
<div style="background: rgba(0,0,255,0.3);">Semi‑transparent blue</div>
RGB Sample RGBA (0.5) Sample
rgb(255,0,0) rgba(255,0,0,0.5)
rgb(0,255,0) rgba(0,255,0,0.5)
rgb(0,0,255) rgba(0,0,255,0.5)

#️⃣ HEX colors

Format: #RRGGBB or #RGB (short). RR,GG,BB are hexadecimal values 00–FF.

<body style="background:#ffaa00;"> ... </body>
<p style="color:#00f;">Blue text (short hex)</p>

🌈 HSL & HSLA colors

hsl(hue, saturation%, lightness%) – hue: 0–360 (0=red, 120=green, 240=blue), saturation: 0%–100%, lightness: 0%–100%.

hsla(hue, saturation%, lightness%, alpha) adds opacity.

<div style="background: hsl(120, 50%, 50%);">Greenish</div>
<div style="background: hsla(240, 80%, 60%, 0.3);">Semi‑transparent blue</div>

📋 HTML forms

<form> container for user input. Attributes: action, method, target, autocomplete, enctype, novalidate.

Basic form example

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Send</button>
</form>

Form attributes table

Attribute Description Example
action URL where data is sent action="/submit"
method GET or POST method="post"
target _self, _blank, _parent, _top target="_blank"
autocomplete on/off autocomplete="off"
enctype encoding for POST enctype="multipart/form-data" (files)
novalidate disable validation novalidate

🎛️ HTML form controls

All input types and elements inside a form.

Text input & password

<input type="text" name="username">
<input type="password" name="pwd">

Radio & checkbox

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

  <option value="nyc">New York</option>

Textarea (multi‑line)

<textarea rows="4" cols="50" name="comment">Default text</textarea>

File upload

<input type="file" name="resume" accept=".pdf,.doc" multiple>

Button types

📽️ HTML multimedia – video, audio, embed, object

Video <video>

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Attributes: autoplay, muted, loop, poster, preload.

Audio <audio>

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
  <source src="song.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Embed <embed> (old but simple)

<embed src="file.pdf" width="500" height="375" type="application/pdf">

Object <object> (more flexible)

<object data="movie.swf" type="application/x-shockwave-flash" width="400" height="300">
  <param name="quality" value="high">
  <!-- fallback content -->
  <img src="fallback.png" alt="alternative">
</object>

<object> can also embed PDF, HTML, images, etc.

📌 Browser support: MP4, WebM, Ogg for video; MP3, WAV, Ogg for audio. Use multiple <source> for compatibility.
